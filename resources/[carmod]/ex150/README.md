# EX150

FiveM addon vehicle resource generated by **CarKit Web**.

## Installation

1. Copy the `ex150` folder to your FiveM server's `resources/` directory.
2. Add `ensure ex150` to your `server.cfg`.
3. Restart your server.

## Spawn Codes

Use the following commands in-game:

`
/spawn ex150
`

## Vehicles

  - ex_150v1_tptgame (Vehicle)

## Requirements

- FiveM server
- No OpenIV required
