# DEFz_demonAG

FiveM addon vehicle resource generated by **CarKit Web**.

## Installation

1. Copy the `defz_demonag` folder to your FiveM server's `resources/` directory.
2. Add `ensure defz_demonag` to your `server.cfg`.
3. Restart your server.

## Spawn Codes

Use the following commands in-game:

`
/spawn defz_demonag
`
