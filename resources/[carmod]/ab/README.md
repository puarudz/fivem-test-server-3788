# AB

FiveM addon vehicle resource generated by **CarKit Web**.

## Installation

1. Copy the `AB` folder to your FiveM server's `resources/` directory.
2. Add `ensure AB` to your `server.cfg`.
3. Restart your server.

## Spawn Codes

Use the following commands in-game:

`
/spawn AB
`

## Vehicles

  - ab_vipp_tptgame (Vehicle)

## Requirements

- FiveM server
- No OpenIV required
